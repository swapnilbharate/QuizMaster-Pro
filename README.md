# 🧠 Advanced Quiz Application (PHP + MySQL)

A full-stack quiz application built using HTML, CSS, JavaScript, and PHP with MySQL integration. The system includes user authentication, dynamic quiz generation, timer-based evaluation, and backend data storage using XAMPP.

## 🚀 Features

* 🔐 User login system (name, email, password)
* 🎯 Randomized quiz questions on each attempt
* ⏳ Timer-based quiz (countdown)
* ✅ Real-time answer validation
* 📊 Score calculation and result display
* 💾 Backend data storage (PHP + MySQL)
* 🔄 Restart quiz functionality

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: PHP
* Database: MySQL
* Server: XAMPP

## 📂 Project Structure

```
quiz-application
 ├── index.html
 ├── styles.css
 ├── script.js
 ├── backend/ (PHP files)
 └── database.sql
```

## ⚙️ Setup Instructions

1. Install and run XAMPP
2. Place project folder in `htdocs`
3. Start Apache & MySQL
4. Import `database.sql` into phpMyAdmin
5. Open in browser:
   http://localhost/quiz-application
